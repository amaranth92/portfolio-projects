## Decody

AI application from Aussie Pus Studio.

Decody translates pet behavior from photos, videos, and audio into understandable human language. It also supports short-form content workflows by helping create subtitles and voice dubbing for clips.

### Role

- Product idea and project planning
- Web/app feature flow design
- AI-assisted behavior interpretation concept
- Subtitle and voice dubbing workflow planning
- Portfolio/public documentation

### Focus

- Make pet behavior easier for owners to understand
- Keep output simple and reusable
- Support photo, video, and audio input scenarios
- Prepare short clip localization features

### Status

In progress. Source code is private while the product direction is still being shaped.

## Photo EXIF Sorter

Python automation tool for organizing large travel photo archives.

Built after a one-year world travel period to classify more than 10,000 photos by date and location metadata.

### Role

- Python script development
- EXIF metadata parsing
- Folder rule design
- Batch file organization

### Focus

- Reduce repetitive manual photo sorting
- Classify images by shooting date and location
- Keep rules adjustable for different archive styles
